\begin{itemize}
\item Support for multiple interface types (selectable): IOb, AXI, AXI-Lite, WISHBONE, APB
\item FIFO only operation
\item Register level and functionality compatibility with NS16550A (but not 16450).
\item Debug Interface in 32-bit data bus mode.
\end{itemize}
